# HTML Authentication System POC

This is a simple HTML-based authentication system proof of concept.

## Pages Included

1. login.html
2. register.html
3. forgot-password.html
4. reset-password.html
5. dashboard.html

## Features

- Basic HTML structure
- Navigation using anchor tags
- No CSS or JavaScript used
- Simple redirection between pages
